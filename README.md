# Student Expense Manager App

> **Coursework 3** > **City St George's, University of London** > **Year 1 Computer Science**

## About
This is a Java-based console application designed to help students manage their finances. It was developed as part of the **Coursework 3** assignment for the Year 1 Computer Science programme.

The application allows users to input various financial data (such as income, rent, and groceries), validates all inputs to prevent errors, and generates a summary of their financial health.
